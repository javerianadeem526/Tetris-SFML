# Tetris Game (C++ + SFML)

A desktop implementation of the classic **Tetris** game developed in **C++** using the **SFML (Simple and Fast Multimedia Library)**. This project was created to strengthen my understanding of game development, object-oriented programming concepts, and event-driven programming.

---

## Features

- Classic Tetris gameplay
- Keyboard controls for movement and rotation
- Collision detection
- Line clearing
- Score tracking
- High score saving
- Game Over detection
- Graphical interface using SFML
- Pause/Resume functionality

---

## Technologies Used

- C++
- SFML

---

## Project Structure

```
Tetris-SFML/
│
├── img/                 # Game images and textures
├── ARIAL.TTF            # Font used in the game
├── functionality.h      # Core game logic
├── pieces.h             # Tetris piece definitions
├── utils.h              # Utility functions
├── highscores.txt       # Stores high scores
├── main.cpp             # Main source file
└── README.md
```

---

## How to Run

1. Install SFML.
2. Open the project in your preferred C++ IDE.
3. Link the SFML libraries.
4. Build and run the project.

---

## Controls

| Key | Action |
|-----|--------|
| Left Arrow | Move Left |
| Right Arrow | Move Right |
| Up Arrow | Rotate Piece |
| Down Arrow | Move Down Faster |
